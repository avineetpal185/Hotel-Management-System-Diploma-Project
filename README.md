# Hotel Management Website

This project is my **Diploma Final Year Project**.
It is a **Hotel Management Website** developed using **HTML, CSS, and JavaScript**.

The purpose of this project is to design a simple hotel website where users can explore hotel services, view rooms, and interact with the website interface.

## Technologies Used

* HTML
* CSS
* JavaScript

## Features

* Hotel homepage design
* Room information section
* Booking interface layout

Diploma Final Year Project
